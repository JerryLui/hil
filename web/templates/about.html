Looper

Contact: Jerry Liu (jerry.x.liu@aptiv.com)
source code

Tasks a log-file to be run through the HIL.

Instructions:
Select a log on the home page to run it through the loop, a login is required. Pressing the Update List button will manually update the file list, the operation will be running in the background and might take some time. Click the desired task id or status to show more detailed info regarding the specific task such as process log and run time. The front page only shows up to 30 tasks, to see a full list of all tasks visit the history page. At your home page there'll be a list of all the tasks run by you, a change password option is there aswell.
